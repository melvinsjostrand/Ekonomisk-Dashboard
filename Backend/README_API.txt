This project uses .NET Framework for API.

The following packages are needed to run API in Visual Studio:
-BCrypt.Net-Next by Chris McKee, Ryan D. Emerl, Damien Miller
-MySql.Data by Oracle
-Swashbuckle.AspNetCore by Swashbuckle.AspNetCore

For the email verification to work you need to enter email, password in appsettings.json.

To set up automated Gmail:
-Have 2-Step authentication
-Generate a password in App passwords
